Traversing Up the DOM Tree

Traversing, which means "move through", are used to "find" HTML elements based on their relation to other elements.

Traversing Upwards - traversing the ancestors (Parent, Grandparent, Great-grandparent .)

Traversing Downwards - traversing the descendants (Child, Grandchild, Great-grandchild etc.)

Sideways -traversing the ancestors the siblings (Brother, sisters available at the same level)

Traversing ancestors -

  • parent()
  • parents()
  • parentsUntil()

Traversing Descendants -

  • children()
  • find()

Traversing filtering -

  • first
  • last
  • eq
  • filter
  • not

Traversing siblings -

siblings() method returns all sibling elements of the selected element. Siblings share the same parent.

  • prev
  • prevAll
  • prevUntill
  • next
  • nextAll
  • nextUntill
